# Reglas de Junie para Poesías Antonio Pardal

Este repositorio utiliza PHP y Composer. Sigue estas reglas:

- **Comandos Unix**: Usa siempre sintaxis de Ubuntu/Unix (ej: `ls`, `cp`, `mv`, `rm`).
- **Ejecución de Tests**:
  `php vendor/phpunit/phpunit/phpunit tests/PoemParserTest.php`
- **Ejecución del Script**:
  `php bin/parse_poem.php input/<fichero.htm>`
  Ejemplos:
  - `php bin/parse_poem.php input/Antonio12.htm`
  - `php bin/parse_poem.php input/Antonio13.htm`
